Let $f: [a,b]\to \mathbb{R}$ be a continuous function. Find all continuous functions $g: [a,b]\to \mathbb{R}$ that satisfy the following equation:
 \begin{equation}\label{E-Q}
 \left(\int_{a}^b g(x)f(x)^2\ dx\right)^2=\left(\int_{a}^b f(x)^2\ dx\right)\left(\int_{a}^b g(x)^2 f(x)^2\ dx\right).
 \end{equation}
