Let $f : [0,1] \rightarrow \mathbb{R}$ be defined as a power series
 $$
 f(x) = \sum_{i=0}^\infty a_i x^i
 $$
 for some $a_i \in \{0,1\}$. Suppose that for some $n \in \mathbb{N}$, we have $f \left( \frac{1}{2^n} \right) \in \mathbb{Q}$. Show that for every odd natural number $q \geq 3$ we have that $f \left( \frac{2}{q} \right) \in \mathbb{Q}$ and if we write $f \left( \frac{2}{q} \right) = \frac{\alpha}{\beta}$ for some integers $\alpha, \beta$ with $\mathrm{gcd}\, (\alpha, \beta) = 1$, then $\beta$ is odd.
