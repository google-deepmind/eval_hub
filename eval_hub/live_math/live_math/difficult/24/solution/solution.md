Notice that $p(1)=a-11+12=a+1$ so the lowest value of $a$ for this expression to be a multiple of $8$ is $a=7$. We will show that this value works for any $n$. Consider $p(n)=7n^5-11n^3+12n$. We will check if $p(n)$ is multiple of $8$ by considering modulus $8$. Notice that for $n \equiv 0,2,4,6 \ \pmod{8}$ it is clear that $p(n) \equiv 0 \ \pmod{8}$. For $n \equiv 1$ we obtain $p(n) \equiv 7-11+12=8 \equiv 0 \ \pmod{8}$ and for $n \equiv 3$ we obtain $p(n) \equiv 7 \cdot 243 -11 \cdot 27 +12 \cdot 3= 3 \cdot(7-11+12) \equiv 0 \ \pmod{8}$. The cases $n=5 \equiv -3$ and $7 \equiv =-1$ are equivalent to the cases $n \equiv 1$ and $n \equiv 3$ so we conclude that $p(n) \equiv 0 \ \pmod{8}$ for any positive integer $n$.
