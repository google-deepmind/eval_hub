Notice that the expression above equals to:
$$\frac{x^4-2x^2y+y^2}{x^2-y}+\frac{x^4+2x^2y+y^2}{x^2+y}=\frac{(x^2-y)^2}{x^2-y}+\frac{(x^2+y)^2}{x^2+y}=2x^2$$
\noindent so the expression does not depend on $y$. The number $2x^2=z$ is an integer if $|x|=+\sqrt{z/2}$. Notice that $\sqrt{z/2} \leq 10$ if and only if $z/2 \leq 100$ so $z \leq 200$. Hence there are $400$ numbers (the positive and the negative ones).
