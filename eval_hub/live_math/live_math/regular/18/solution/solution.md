We use the fact that if $z$ is a complex root, then also $\bar{z}$ is a complex root. Let $z = x + iy$ for some $x, y \in \mathbb{R}$. Then we have $3z+1 + i = \tilde{z}$, so $3x + 3iy + 1  +i = x-iy$. We have $2x+1 + i(4y +1) = 0$, so $x = -\frac{1}{2}$ and $y=-\frac{1}{4}$. Thus the roots are $z = -\frac{1}{2} -\frac{1}{4}i$ and $\tilde{z} = -\frac{1}{2} + \frac{1}{4}i$. Now we can use Viete formulas, namely $-\frac{a}{2} = -1 -\frac{1}{2} - \frac{1}{4}i-\frac{1}{2} + \frac{1}{4}i = -2$, so $a=4$. Then we have $-\frac{c}{2} = -1(-\frac{1}{2}-\frac{1}{4}i)(-\frac{1}{2}+\frac{1}{4}i) = (\frac{1}{4}i + \frac{1}{2})(\frac{1}{4}i-\frac{1}{2}) = -\frac{1}{16}-\frac{1}{4}=-\frac{1}{16} - \frac{4}{16} = - \frac{5}{16}$. So $c = \frac{10}{16}= \frac{5}{8}$. Now, using the fact that -1 is a root, we have $-2+a-b+c 
 =0$, so $-2 + 4 - b + \frac{5}{8}=0$, and thus $b = \frac{21}{8}$. So $a^2 + 8(b+c) = 16 + 21 + 5 = 42$.\\
