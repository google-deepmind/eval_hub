From the equation $\frac{12x-524y}{3y-2x}=500$ we get
    \begin{align*}
        12x-524y &= 1500y - 1000x\\
        1012 x &= 2024y\\
        x &= 2y.
    \end{align*}
    So $\frac{2012x-2030y}{2030x + 2012y} = \frac{2012x - 2030\cdot \frac{1}{2}x}{2030x + 2012 \cdot \frac{1}{2}x} = \frac{2012x - 1015x}{2030x + 1006x} = \frac{997}{3036}$. The last fraction is irreducible, thus $b-a = 2039$.
