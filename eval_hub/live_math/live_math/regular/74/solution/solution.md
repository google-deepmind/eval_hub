Notice that
 
 $$117649a^6 + 14406a^4b^2 + 8b^6 = 
 117649a^6 + 14406a^4b^2 + 588a^2b^4 + 8b^6 - 588 a^2b^4 = (49a^2 + 2b^2)^3 - 588a^2b^4.
 $$
 
 The $AM-QM$ inequality for the numbers $7a$, $b$, $b$ gives us
 $$\sqrt{\frac{49a^2 + 2b^2}{3}} \ge \frac{7a+2b}{3} = \frac{35}{3}.$$
 Hence $49a^2+2b^2 \ge \frac{35^2}{3}.$
 
 The $AM-GM$ inequality for the same numbers will give us 
 $$\sqrt[3]{7ab^2} \le  \frac{7a+2b}{3} = \frac{35}{3}.$$
 This means that 
 $$588a^2b^4 = 12\cdot 49a^2b^4 \le 12\cdot\left(\frac{35}{3}\right)^6. $$
 Summing up, we get
 
 $$(49a^2 + 2b^2)^3 - 588a^2b^4 \ge \left(\frac{35^2}{3}\right)^3 - 12\cdot \left(\frac{35}{3}\right)^6 = 35^6\cdot \left(\frac{1}{27} - \frac{12}{729}\right) = \frac{35^6\cdot(9 - 4)}{243}
 =  \frac{35^6\cdot 5}{243}.
 $$
 This is a reduced fraction, the numerator is equal to 9191328125, and hence the answer is 9191328125+243 = 9191328368.

Note that we get an equality for $7a=b$, i.e. for $a = 5/3$ and $b=35/3$.
