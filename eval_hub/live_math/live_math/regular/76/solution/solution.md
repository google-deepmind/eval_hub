Notice that the expression is equivalent to:
$$\frac{23}{29}-\frac{1}{2024} < \frac{n}{3^m} < \frac{23}{29} \ \mbox{ if and only if } \ \left(\frac{23}{29}-\frac{1}{2024}\right)3^m< n< \frac{23}{29} \cdot 3^m$$
\noindent and since $\left(\frac{23}{29}-\frac{1}{2024}\right)3^m$ and $\frac{23}{29} \cdot 3^m$ cannot be integers, the existence of $n$ is sure if the length of the corresponding interval is greater than $1$, that is:
$$\frac{3^m}{2024} > 1 \ \mbox{ if and only if } m >6.$$


\noindent Take $m=7$. Then:
$$\left(\frac{23}{29}-\frac{1}{2024}\right)3^7< n< \frac{23}{29} \cdot 3^7$$
\noindent is equivalent to $1733.44 < n < 1734.52$ so $n=1734$ would be good. We check $m \leq 6$ and the only one that allows the existence of an integer $n$ is $m=6$: $577.812 < n < 578.172$. Hence, the lowest value of $m$ is given by $6$ since we get $n=578$. The answer is $m=6$.
