Notice that by the triangle inequality we have that $b+c > a$ so $a \leq 10$ (if $a \geq 11$ then $b+c \leq 10$, a contradiction). Hence:
\begin{itemize}
\item For $a=10$ we have $b+c=11$ so $(b,c)=(9,2), (8,3), (7,4), (6,5)$.
\item For $a=9$ we have $b+c=12$ so $(b,c)=(8,4), (7,5)$.

\item For $a=8$ we have that $b+c=13$ so $(b,c)=(7,6)$. 
\end{itemize}

We have $4+2+1=7$ scalene triangles.
