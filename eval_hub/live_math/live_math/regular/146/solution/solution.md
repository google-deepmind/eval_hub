Notice that $7/8-5/6=2/48=1/24$. Consider first that $5/6 < x <  7/8$. Then:
$$\frac{5}{6}+\frac{2}{3} \cdot \left(\frac{7}{8}-\frac{5}{6} \right)=\frac{31}{36}.$$

\noindent Notice that it is not possible that $x < 5/6$ so we consider $x > 7/8$. Then:
$$x=\frac{7}{8}+\frac{1}{24}=\frac{22}{24}=\frac{11}{12}.$$

\noindent Hence there are two solutions: $31/36$ and $11/12$. The larger is $11/12$, so the answer is $23$.
